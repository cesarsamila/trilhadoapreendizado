2. del:Representa uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

Esse texto foi deletado

3. ins:Representa um intervalo de texto que foi adicionado a um documento.

Este texto foi inserido

4. sub:O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor.

Esse texto contem subscript text abaixo.

5. sup:O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor.

Esse texto contem superscript text acima.

6. blockquote:Indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo (ver Notas sobre como mudá-lo). A URL para a fonte da citação pode ser dada usando o atributo cite

Esta é uma citação tirada da Google Classroom.

7. q:Indica que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo

According to Mozilla's website, Crome 2.0 lançou suas ferramentas em 2010.

10. cite:Representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada de acordo com as convenções usadas para a adição dos metadados de citação.

Mona lisa de Leonardo da Vinci Data 1503-1506.
